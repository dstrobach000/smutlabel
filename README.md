# smutlabel

Next.js site for **SMUT** / smutlabel — App Router, TypeScript, and Tailwind CSS v4.

## Prerequisites

- Node.js 20+ recommended
- npm (ships with Node)

## Scripts

```bash
npm run dev    # local dev server — http://localhost:3000
npm run build  # production build
npm run start  # run production server (after build)
npm run lint   # ESLint
```

## Repository

Remote: [github.com/dstrobach000/smutlabel](https://github.com/dstrobach000/smutlabel)

Initial push (after GitHub repo exists and you are authenticated):

```bash
git remote add origin https://github.com/dstrobach000/smutlabel.git   # if not set
git push -u origin main
```

## Project layout

- `app/page.tsx` — homepage
- `app/layout.tsx` — root layout and metadata
- `app/globals.css` — global styles and Tailwind import
